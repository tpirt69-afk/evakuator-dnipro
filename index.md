# Промпт для создания сайта — Enterprise Grade

## Инструкции
Замени все [ПЛЕЙСХОЛДЕРЫ] на данные своего бизнеса перед использованием промпта.

## Данные бизнеса (всего 3 параметра)
- **Название бизнеса:** [НАЗВАНИЕ_БИЗНЕСА]
- **Индустрия:** [ИНДУСТРИЯ]
- **Цвет бренда:** [ЦВЕТ_БРЕНДА]

## Директива по генерации контента

Ты получишь ТОЛЬКО три значения выше. На основе названия бизнеса, индустрии и цвета бренда ты должен самостоятельно сгенерировать ВСЁ содержимое сайта. Включая, но не ограничиваясь:

### 1. Hero-секция

**Макет:** Центрированное выравнивание, каскадная анимация появления

**Контент:**
- Цепляющий заголовок (5-12 слов) и поддерживающий подзаголовок (1-2 предложения)
- Название продукта должно быть на видном месте
- Стиль: профессиональный, современный, соответствующий индустрии
- Основная кнопка CTA и вторичная ссылка (например, "Подробнее", "Смотреть цены")

**Визуальные элементы:**
- Качественное изображение или градиентный фон, подходящий индустрии
- Текст должен быть читаемым с правильным контрастом

### 2. Секция преимуществ

**Макет:** Сетка 2-3 колонки, равномерные отступы

**Контент:**
- 3-5 ключевых преимуществ бизнеса
- У каждого преимущества: убедительный заголовок и описание
- Релевантная иконка для каждого пункта

**Стиль:**
- Профессиональная типографика
- Единообразные отступы и выравнивание
- Цвет бренда как акцент

### 3. Социальное доказательство / Логотипы

**Макет:** Полноширинная, центрированная секция с горизонтальным расположением логотипов

**Контент:**
- Секция "Нам доверяют" или "О нас пишут"
- Логотипы клиентов, партнёров или изданий

**Стиль:**
- Логотипы на 50% прозрачности, в оттенках серого
- При наведении логотипы становятся цветными
- Горизонтальное центрирование, равномерные отступы

### 4. CTA-секция / Прайс

**Макет:** Заметная, полноширинная секция

**Контент:**
- Понятный призыв к действию
- Информация о ценах (если применимо)
- Форма записи или контактная форма

**Стиль:**
- Цвет бренда как основной фон
- Высокий контраст для читаемости
- Адаптивность под мобильные

### 5. Футер

**Макет:** Полноширинный футер с организованными разделами

**Контент:**
- Информация о компании
- Навигационные ссылки
- Контактная информация
- Ссылки на соцсети
- Копирайт

**Стиль:**
- Профессиональный, чистый дизайн
- Соответствие цветовой схеме бренда

## Требования к дизайну

- **Адаптивный дизайн:** корректная работа на мобильных, планшетах и десктопе
- **Анимации:** плавные анимации появления и hover-эффекты
- **Единство бренда:** цвет бренда используется во всех секциях
- **Типографика:** профессиональные, читаемые шрифты с правильной иерархией
- **Доступность:** правильные коэффициенты контрастности и семантический HTML
- **Производительность:** оптимизация для быстрой загрузки

## Формат вывода

Сгенерируй полностью готовый к продакшену сайт:
- Чистый, современный дизайн
- Все секции полностью реализованы
- Адаптивная вёрстка
- Плавные анимации
- Стилизация в соответствии с брендом
- Готово к деплою

## Инструкция по использованию

1. Замени [НАЗВАНИЕ_БИЗНЕСА] на название своей компании/продукта
2. Замени [ИНДУСТРИЯ] на индустрию своего бизнеса
3. Замени [ЦВЕТ_БРЕНДА] на основной цвет бренда (например, #FF6B6B, #4ECDC4)
4. Вставь весь промпт в Claude Code
5. Claude Code сгенерирует полноценный сайт
